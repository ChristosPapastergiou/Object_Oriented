# Object-Oriented
Programming in object-oriented style and simulating some every day actions

-------------------------------
How to compile and run
-------------------------------
c++  :

	compile : make
	run     : ./Project arg1 arg2 arg3 arg4
  
java :

	compile : javac Hierarchy.java  
	run     : java Hierarchy arg1 arg2 (arg3)
  
Have fun.
